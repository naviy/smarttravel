Подключение к GALILEO

Необходимо на каждом терминале выполнить следующие команды.

Команды линковки (вводить один в один):
HMLMFE3267DA
HMOMFE3267-U

Команда проверки:
HMLD  

Должно выдать:
CRT      TKT DEV  ST FM / ITN DEV  ST FM / MIR DEV  ST FM /  
FE88AC    E686C0D  U  T    CD4347D  U  IE   FE3267D  U
(в конце 3267 U)

Если команда HMLMFE2168DA выдаёт INVALID GTID, то звонить в техподдежку Galileo с просьбой, 
чтобы они прописали девайс для данной зоны (сказать им ID терминала по команде +J)

После этого выслать нам коды офисов PCC.


Команда для перевыпуска: tkpdad
https://support.travelport.com/webhelp/RussianKB/files/tkpdad.htm?v=3
