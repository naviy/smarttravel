Для подключения пультов нужно ещё на каждом запустить команды линковки:

Например, офис 3WT0:

HMLMFE3267DA
HMOMFE3267-U

Система должна выдать следующий ответ о подключении: 
DEVICE FE3267 IS NOW UP

Если после этого не будут приходить файлы с какого-то пульта, то нужно звонить Виктору Кузьмину
